# React Native 운동 SNS app

> SNS 클론 코딩 강의 내용을 기반으로 React Native 앱 구현해보는 과제
 
## 개발 환경

This proejct bootstrapped using [`@react-native-community/cli`](https://github.com/react-native-community/cli).

#### 화면 구성과 관련한 라이브러리
 - react-native-modal
 - react-navigation
#### 카메라 앨범 관련 라이브러리
 - react-native-camera-roll
